📘 Instrucciones del Proyecto

Este proyecto es un CRM Agrícola que permite a una empresa del sector gestionar de forma eficiente sus relaciones con clientes, empresas y productos, facilitando el seguimiento de interacciones y el análisis de datos.

Está diseñado para que distintos tipos de usuarios (superadministradores, supervisores y vendedores) puedan trabajar colaborativamente, respetando niveles de permisos según su rol.

🧩 Tecnologías Utilizadas

  • Backend: Python + Django
  • Frontend: HTML5, CSS3, Bootstrap, JavaScript, Chart.js
  • Base de Datos: SQLite3
